# PR2 Mock Data (Reference Only)

These files (`mockApi.js` + the `*.json` files) are what PR2's Interim
Build used before the real backend existed. They are **no longer
imported anywhere** in PR3 — `src/data/api.js` replaced `mockApi.js`
once the PHP Slim 4 backend was ready.

Kept here only so you can see what changed between PR2 and PR3, and as
a fallback if you ever want to demo the UI without the backend running.
Safe to delete before final submission if you want a cleaner repo.
